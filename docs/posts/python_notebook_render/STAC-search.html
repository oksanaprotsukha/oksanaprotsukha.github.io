Catalog search

We can narrow the search within the catalog by specifying a time range, an area of interest, and the collection name. The simplest way to define the area of interest to look in the catalog are:

  • a GeoJSON-type dictionary with the coordinates of the bounding box,
  • as a list [xmin, ymin, xmax, ymax] with the coordinate values defining the four corners of the bounding box.

You could also use a point, or some more complex polygon.

In this lab we are going look for NAIP scenes over Santa Barbara from 2018 to 2020. We’ll use GeoJSON method to define the area of interest:

Code 
# temporal range of interest
time_range = "2018-01-01/2023-01-01"

# NCEAS bounding box (as a GeoJSON)
bbox = {
    "type": "Polygon",
    "coordinates":[
        [
            [-119.70608227128903, 34.426300194372274],
            [-119.70608227128903, 34.42041139020533],
            [-119.6967885126002, 34.42041139020533],
            [-119.6967885126002, 34.426300194372274],
            [-119.70608227128903, 34.426300194372274]
        ]
    ],
}

# catalog search
search = catalog.search(
    collections=['naip'],
    intersects=bbox,
    datetime=time_range)
